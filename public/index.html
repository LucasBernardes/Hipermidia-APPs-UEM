Caracteristicas dos aplicativos

O projeto "Desenvolvimento de aplicativos para dispositivos móveis para automação de serviços oferecidos aos alunos e servidores da UEM.", tem em seus aplicativos seus objetivos:

Apps Android

Desenvolvimento de aplicativos para a plataforma android através do framework ionic.

Atualização

Os aplicativos desenvolvidos estão sempre em constante atualização, afim de manter os aplicativos funcionando.

Design limpo

Aplicativos são desenvolvidos sobre a constante avaliação das interfaces, bucando sempre simplificar e agradar os usuários

Amigável ao usuário

O desenvolvimento dos aplicativos segue regras ergonômicas para garantir a alta usabilidade dos apps.

Testes

O desenvolvimento conta com a ajuda de usuários testers de diversos cursos da uem, que proporcionam feedback importantes para a interfaciação.

Otimizados

Buscamos o desenvolvimento de aplicativos ágeis e leves, que respondem bem aos diversos aparelhos do mercado.

Código

Desenvolvido com código limpo e bem comentado.

Especializado para a UEM

Aplicativos feitos especialmente para as a Universidade Estadual de Maringá e seus usuários.

Encontre na AppStore e Marketplace

Encontre os aplicativos nas diversas plataformas de download, dos principais sistemas operacionais.

Projeto

O objetivo deste projeto é o desenvolvimento de aplicativos para dispositivos móveis, utilizando a metodologia Scrum, que automatizem procedimentos e serviços ofertados para a comunidade acadêmica e administrativa da UEM.
